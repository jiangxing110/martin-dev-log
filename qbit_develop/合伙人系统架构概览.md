# 合伙人系统架构概览

## 系统定位

Qbit 合伙人（Partner）系统是面向渠道推广和合作伙伴的激励管理平台，覆盖**合伙人注册 → 客户引流 → 业务提成计算 → 提现审核 → 结算对账**的完整链路。

## 跨项目分布

合伙人功能横跨两个项目：

| 项目 | 语言/框架 | 职责 |
|------|----------|------|
| `qbitpay_service` | NestJS + TypeScript + GraphQL | 合伙人前端运营系统：注册、订单管理、审核流、数据展示 |
| `qbit-assets` | Spring Boot 3 + Java 17 | 合伙人后端管理系统：Admin/Merchant API、定时任务、数据分析 |

---

## 三大核心概念

### 1. 合作伙伴（Business Partner / Supplier）
- 表 `partners`，由 `common_all/partner/manage/` 模块管理
- 类型：`partners`（合作伙伴）或 `supplier`（供应商）
- 用于生态页面展示，支持多语言、logo、官网链接，在线/下线发布
- 与返佣体系解耦，属于 CMS 类功能

### 2. 合伙人返佣（Referral Partner Commission）
- 表 `"partnerOrder"`，由 `common_all/partner/merchant/` 和 `qbitpay_service` 的 `modules/partner/` 共同管理
- 核心业务：合伙人通过邀请码（ReferralCode）引流客户
- 客户完成 KYC/KYB、激活、结汇、充值等动作后，系统自动生成提成订单
- 提成可提现，提现需经过 CEO 审核 → 财务审核流程
- **2025-04-18 政策调整**：国内版 QBIT 部分业务提成分销，保留海外版 QbitInternational

### 3. 渠道方返佣（Channel Commission）
- 表 `"channelCommission"`，由 `common_all/partner/commission/` 模块管理
- 按季度/月度周期结算的渠道返佣
- 包含趋势分析、员工交易量分析等
- 由定时任务 `ChannelCommissionJob` 驱动

## 合伙人账户类型

系统通过 `AccountGeneralTypeEnum.Channel`（'合伙人'）标识合伙人账户。
合伙人账户在账户体系中被标记为 `type = Channel`，拥有自己的 `referralCode` 邀请码。

### 系统版本区分

合伙人分为两个系统版本：
- **QBIT（国内版）**：人民币结算，结汇（全球账户付款）、量子账户充值等
- **QbitInternational（海外版/web3版）**：美元结算，Vendor 订单、加密资产等

## 合伙人提成计算逻辑

### 计算公式摘要

1. **全球账户入金手续费分成**：`fee * 25% - amount * 0.001 * 25%`（国内版，按 CNY 转换）
2. **全球账户结汇分成**：`(收取费用 / 结汇金额 - 0.1%) * 结汇金额(CNY) * 25%`
3. **量子账户充值阶梯**：>=50 万 USD 返 0.2%，>=80 万返 0.4%，>=100 万返 0.6%，每 50 万返一次 0.6%
4. **web3 量子账户充值**：`fee * 0.1`
5. **海外版公司注册**：按国家/套餐固定金额（HK $63.92、US Regular $39.93、US Premium $10）
6. **海外版全球账户开户费**：`usdAmount * 0.5`

### 一年有效期内

从客户账户创建之日起一年内的业务才产生提成（`isWithinOneYear` 逻辑）。

## 提现审核流程

```
待审核(Pending)
    ↓ CEO 审核（飞书审批）
CEO 审核通过(CEOReviewPassed)
    ↓ 财务审核（飞书审批）
财务审核通过(FinanceReviewPassed)
    ↓ 打款中(Processing)
    ↓
已打款(Closed)
```

任意节点可驳回：`Rejected → Fail`

## 跨系统交互

`qbitpay_service` 与 `qbit-assets` 之间通过 BridgeService（HTTP）进行交互：
- `POST /core/partner/order/add/commission` - 指派合伙人补提成通知
- `POST /core/partner/order/paymentFee` - 查询提现手续费
- `GET /assets/exchange/exchange/estimate-quote` - 询价
- `GET /assets/withdrawal/currencies` - 提现币种
